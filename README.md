# ❖ D{0-1}KP 智能求解分析系统
本项目是针对 **D{0-1}KP (Dimension 0-1 Knapsack Problem)** 深度定制的综合求解与可视化平台。系统集成了动态规划精确求解算法、大规模数据集处理、现代响应式 GUI 交互以及企业级数据导出功能，旨在为复杂的背包优化问题提供直观、高效的软件级解决方案。

## ✨ 核心特性 (Core Features)
* **🧠 核心算法引擎**：采用经典的 **动态规划 (Dynamic Programming)** 策略，确保求解结果的绝对准确性。内置按“价值/重量比”非递增排序的预处理模块。
* **🎨 现代交互式 UI**：
* 引入 **FlatLaf** 现代扁平化主题，彻底重构原生 Java Swing 界面。
  * 采用 `JSplitPane` 实现极客风控制台与操作面板的**拖拽式无缝响应布局**。
  * 内置友好的系统启动引导语 (User Onboarding) 与按钮状态防呆设计。
* **📊 高级数据可视化**：基于 `JFreeChart` 引擎，渲染带有透明度、科技色调的半透明点阵散点图，直观呈现物品重量与价值的分布规律，支持鼠标悬浮探针与框选放大。
* **📑 企业级报表导出**：除基础的 `.txt` 文本输出外，引入 **Apache POI** 组件，支持一键将求解报告、运行耗时及选中的物品明细导出为排版精良的 `.xlsx` (Excel) 格式表格。

## 🛠️ 技术栈与依赖 (Tech Stack)
* **开发语言**: Java (JDK 8+)
* **构建工具**: Maven
* **UI 框架**: Java Swing + FlatLaf 3.4.1
* **图表引擎**: JFreeChart 1.5.3
* **文档流处理**: Apache POI 4.1.2 (Excel OOXML)
