# ❖ D{0-1}KP 智能求解分析系统
本项目是针对 **D{0-1}KP (Dimension 0-1 Knapsack Problem)** 深度定制的综合求解与可视化平台。系统集成了动态规划精确求解算法、大规模数据集处理、现代响应式 GUI 交互以及企业级数据导出功能，旨在为复杂的背包优化问题提供直观、高效的软件级解决方案。

## ✨ 核心特性 (Core Features)
* **🧠 核心算法引擎**：采用经典的 **动态规划 (Dynamic Programming)** 策略，确保求解结果的绝对准确性。内置按“价值/重量比”非递增排序的预处理模块。
